Arrays vs Hashes, broken down!

I'll attempt to explain these concepts as clearly as I can, to an audience with little prior knowledge of these concepts!

    • Some context: What are arrays and hashes and when would we use them? (with examples!)

    So let's start with an array. You can think of an array, as a kind of filing cabinet for things! Let's imgaine we have a filing cabinet that we use to "store" the names of different reptiles. We can choose how many reptiles names we would like to store in the cabinet. When we open the cabinet up, there are an infinite amount of "slots" (or indexes) for us to store our reptile names. So we want to make our first entry into the cabinet with the word "turtle". Logically we make our first entry in the first slot or index. Then after we do this, we want to store the word "lizard" in the reptile filing cabinet, we store this word in the index after turtle for our second entry. This process continues until we have as many reptiles names as we would like in our filing cabinet!

    Now let's consider an array in a similar manner. An array is an object which can "store" various entries in it's slots or "indexes" just like a filing cabinet. You can then access to names which you have placed in your array whenever you would like. In a way it's a bit like the filing cabinet described above! Before going on I'll quickly show you what an array looks like in Ruby:

    #code snippet reptiles = ['turtle', 'lizard', 'reptar', 'komodo dragon']

    So there you have it, an array is used to store data which can be accessed and manipulated once we know some techniques which I shall discuss later. An array can be used to store data about words (or 'strings'), it can store numbers, it can store boolean values (true/false), or it can even store other arrays, or hashes (which we shall dicuss later.) An object can be stored in an array.

    Accessing data within an array!

    Now that we have a basic understanding of arrays it's time to start playing with them! We can access an item in an array by typing in it's index number, as follows: reptiles[3] = 'Komodo Dragon' reptiles[1] = 'Lizard' (note that index 1 referes to the second item, since counting starts at 0!) Before I want to talk about iterating over array's let's mention what a hash is.

    Hashes

    Hashes are similar to arrays in the sense that they are both used to store data. However a hash uses key / value pairs. This means that within a hash there are a bunch of keys, which all points to different values. Here's what this looks like: #code snippet: A hash of multiple items with keys. Each key can only be typed in once though, however a value can be typed in multiple times. However once you typed in those two fighters names as keys, you can not use those two keys again.
